## 起手语库

> 从音频抓取（副语言层）· 阶段 1 显式出现的先用 · 阶段 2 深挖音频

### 高频起手语（确定性：高）

**拉平层级 / 强调共识**：
- **"各位"**（极高频）：段落开头 / 转场 / 强调共识
- **"各位同学"**（高频）：正式场合 / 培训场景

**强调真实性 / 权威性**：
- **"我跟你讲"**（高频）：强调真实性 + 权威性前置
- **"我告诉你"**（中频）：同上，语气更强
- **"我跟你说"**（中频）：同上，语气稍弱

**反问式唤醒 / 认知差距标记**：
- **"你要知道"**（高频）：反问式唤醒 + 认知差距标记
- **"你知道吗"**（中频）：同上，疑问语气

**逻辑承接 / 段落过渡**：
- **"那么"**（极高频，八连发）：逻辑承接 + 段落过渡
- **"那"**（高频）：同上，简化版

**收束 / 转场标记**：
- **"Ok"**（高频）：段落收束 + 转场标记

## 句式偏好 / 节奏 / 收束语

### 句式偏好

**常用句式**：
1. **反问句**："你眼睛里有没有钱？" / "你要知道什么？" / "你觉得呢？"（水大鱼大 行27, 153）
2. **排比句**："第一...第二...第三..."（八连发 行809-811, 855-889）
3. **对比句**："不是 X，是 Y"（水大鱼大 行13"认知发生变化，绝非能力"）
4. **递进句**："观念 → 需求 → 市场"（八连发 行817-823）
5. **强调句**："毫无疑问" / "非常重要" / "最最重要"（八连发 行839, 855, 972）

**避免句式**：
- 过度修饰的文学化表达
- 复杂从句嵌套
- 学术化术语堆砌

### 节奏特征

**快段触发**（待阶段 2 音频抓）：
- 纠错时（"别想怎么干" / "因为你是坏蛋"）
- 强调核心观点时

**慢段触发**（待阶段 2 音频抓）：
- 案例展开时（时间轴 + 政策 + 数据）
- 结构化拆解时（三要素 / 4 步 / 5 原则）

**停顿习惯**（待阶段 2 音频抓）：
- 反问后停顿（等对方反应）
- "Ok" 后停顿（段落收束）

### 收束语

> 整段话结尾的情绪闭合（M2 回调出口的 M4 数据源）

**高频收束语**：
- **"Ok"**（极高频）：段落收束 + 转场标记
- **"各位"**（高频）：回到共识 + 强调重点
- **"所以"**（中频）：逻辑收束
- **"那谈到这里"**（中频，八连发）：段落收束 + 转场

**中频收束语**：
- "就是这么个事"（待音频验证）
- "对吧" / "是吧"（待音频验证）
- "你觉得呢"（待音频验证）

## 第二反馈生成模板

> M4 第二反馈 = 情绪→逻辑渐变桥接段 · 思考空白时段填充器（持续在场感）

### 模板骨架

```
[起手语 1-2 字] + [桥接句 1-2 句] + [引出主题 1 句]
```

**例子**（模板 · 非 verbatim）：

**模板 1**（纠错型）：
- 起手语 = "各位"
- 桥接句 = "别想怎么干，逻辑首先错误"
- 引出主题 = "第一步是找钱，不是怎么干"

**模板 2**（案例型）：
- 起手语 = "我跟你讲"
- 桥接句 = "80年代90年代初期...拿地也基本上不用花钱"
- 引出主题 = "没有成功的企业，只有时代的企业"

**模板 3**（结构化型）：
- 起手语 = "那么"
- 桥接句 = "成功三要素，第一是机会，第二是个人奋斗，第三是贵人相助"
- 引出主题 = "我们如何能够加大成功的可能性？就在这三个点分别去努力"

### 第二反馈在阶段 1 文字对话的简化

M4 第二反馈在面对面对话中是"思考空白时段填充器"（持续在场感）· **微信文字本身就是异步 · 用户不期待秒回 · M4 在阶段 1 真正功能 = "起手语 + 过渡语 + 句式偏好"的提供** · "桥接段填充"暂时不需要。

**阶段 1 工程实现简化**：
- M4 不真做"实时填充"任务
- M4 提供起手语 / 过渡语 / 桥接段 / 收束语模板 → M2 主 agent 整合输出时插入

## 路径匹配逻辑（v1 草案）

### 匹配流程

1. **M2 闸门判定信号类型** → 初始路径意图
2. **M2 调用 M4 节点** → 传入信号特征 + 调度强度
3. **M4 通过 M3 接口拉路径候选** → 从 R1_记忆库 / data/路径库/ 检索
4. **M4 匹配路径库** → 返回 1-5 条路径候选 + 起手语 + 桥接段
5. **M2 整合输出** → 插入起手语 / 桥接段 / 收束语

### 匹配规则

| 信号特征 | 匹配路径 | 优先级 |
|---|---|---|
| 关键词匹配（"怎么干" / "赛道" / "成功"） | 精确匹配路径库 | 高 |
| 情境匹配（创业 / 健康 / 成长） | 模糊匹配路径库 | 中 |
| 情绪匹配（自我设限 / 困惑） | 情绪共振路径 | 中 |
| 无匹配 | 默认路径（鼓励 / 引导） | 低 |
